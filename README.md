# sql-data-warehouse-end-to-end
A comprehensive SQL-based data warehousing solution utilizing Medallion Architecture (Bronze, Silver, Gold) to transform raw ERP/CRM data into actionable business intelligence using T-SQL and Star Schema modeling.

#Technical Stack & Skills Demonstrated

Database: Microsoft SQL Server (SSMS)

Language: T-SQL (Advanced Joins, CTEs, Window Functions, DDL/DML)

Architecture: Medallion Architecture (Bronze/Silver/Gold layers)

Modeling: Dimensional Modeling (Star Schema), Primary/Foreign Key Relationships

ETL: Data Cleaning, Type Casting, and Automated Ingestion Logic
